# adsmn-face-segment
## Installation

### Python Dependencies

```bash
pip install -r requirements.txt
```

### Running Flask Server

Navigate to the `src` folder and run the Flask server using:

```bash
python flask_app.py
```

### Node Server

Run the Node server using:

```bash
npm start
```
